** 構築方法
1. ~docker build -t learnpyai .~ でビルドする。
2. Kitematicで起動し、以下の設定とする。
   - Port: 8888:8888
   - Volume: /here-folder/notebooks:/notebooks
# ~docker run -it --name learn-py-ai -v /here/folder/notebooks:/notebooks -p 8888:8888 learnpyai~
